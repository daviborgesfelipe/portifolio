E-Agenda Medica

Info
Descrição :

Desenvolvimento de um aplicativo web para eficiente gestão do cronograma de atividades em clínicas médicas. Recursos chave incluem agendamento de cirurgias e consultas, identificação única de médicos via CRM, definição de horários, automação de períodos de recuperação, e flexibilidade para eventos passados ou futuros, com opções de modificação e exclusão. Funcionalidades adicionais abrangem alertas de conflitos de horários para o mesmo médico e uma lista dos 10 médicos mais ativos em determinado período.

Tecnologias Utilizadas :

O E-Agenda Médica é uma aplicação moderna com Angular no frontend para criar SPAs responsivas. No backend, utiliza ASP.NET Core Web API e Entity Framework Core para operações eficientes no banco de dados. Adota práticas modernas, frameworks como FluentValidation garantem integridade e transparência, e é organizado para facilitar manutenção e expansão. Este projeto representa a convergência de tecnologias avançadas, proporcionando uma solução eficaz e moderna para a gestão de cronograma em clínicas médicas.
