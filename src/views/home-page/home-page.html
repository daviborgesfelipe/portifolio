E-Agenda Medica

Info

Descrição :

Desenvolvimento de um aplicativo web para eficiente gestão do cronograma de atividades em clínicas médicas. Recursos chave incluem agendamento de cirurgias e consultas, identificação única de médicos via CRM, definição de horários, automação de períodos de recuperação, e flexibilidade para eventos passados ou futuros, com opções de modificação e exclusão. Funcionalidades adicionais abrangem alertas de conflitos de horários para o mesmo médico e uma lista dos 10 médicos mais ativos em determinado período.

Tecnologias Utilizadas :

O E-Agenda Médica adota tecnologias modernas para oferecer uma experiência eficiente.

No frontend, utiliza Angular para construir Single Page Applications e RxJS para programação assíncrona. Angular Material e Bootstrap são empregados para criar interfaces modernas. A estrutura do projeto segue as melhores práticas, com componentes reutilizáveis, serviços de lógica de negócios, modelos de dados e views.

No backend, faz uso do ASP.NET Core Web API, Entity Framework Core para mapeamento objeto-banco de dados, e ferramentas como FluentValidation, FluentResults, AutoMapper, Serilog e SequentialGuidGenerator para aprimorar a qualidade e eficácia do desenvolvimento.
